# Completando as Operações Matemáticas

Este é um jogo web simples para praticar operações matemáticas.

## Como Jogar

1.  Abra o arquivo `index.html` em seu navegador.
2.  Clique no botão "Iníciar".
3.  Selecione o nível de dificuldade: Fácil, Médio ou Difícil.
4.  Escolha o número de jogadores (de 1 a 4).
5.  Digite os nomes dos jogadores.
6.  Responda o máximo de questões matemáticas que conseguir antes que o tempo acabe.
7. Ao fim será exibido a pontuação de cada jogador e o vencedor, quando hover.



## Níveis

*   **Fácil:** Operações de adição apenas.
*   **Médio:** Operações de adição e subtração.
*   **Difícil:** Operações de adição, subtração, multiplicação e divisão.
